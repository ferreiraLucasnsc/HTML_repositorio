Inspecionador de Elementos

(Voltar ao topo)

A ferramenta de inspecionar elementos está disponível na maioria dos navegadores, incluindo Chrome, Safari e Firefox. Sua função é disponibilizar para o usuário a visualização do código-fonte de uma página web, além de permitir temporárias mudanças em elementos do site, mantendo o código-fonte intacto.

Este instrumento permite testar alterações, diagnosticar problemas, entender certos recursos de sites observando as classes CSS e até simular um site em dispositivos móveis.

Estrutura básica do HTML

(Voltar ao topo)

Um arquivo HTML possui uma estrutura básica que é interpretada por grande parte dos navegadores. Esta estrutura é dividida em três partes:

  • tag <HTML>: tag que vem antes de todo o código (exceto quando é preciso por a tag <"!DOCTYPE HTML">, que declara o tipo de arquivo para a interpretação do navegador). Essa tag possui todos os elementos do documento, e todos os documentos HTML precisam ter esta tag no início e fim.
  • tag <head>: cabeçalho da sua página, a head possui informações que não são totalmente vistas pelo usuário, como metadados de tipo de caracteres. O título, também introduzido por tag, é visível na aba aberta. Estas informações servem para esclarecer ao navegador funções importantes para sua página.
  • tag <body>: o corpo do seu texto, a tag body é tudo o que é visível para o usuário. Elementos como textos, arquivos multimídia e estilos por CSS estão no corpo da sua página HTML.

Tags

(Voltar ao topo)

As tags são usadas para informar ao navegador a estrutura do site, sendo elas interpretadas para exibição. Sua principal característica é uma palavra dentro dos sinais de chevron, ou seja, "maior que" e "menor que" (<>).

Os dois tipos de tags são as fixas, que precisam apenas de uma tag para funcionarem (como por exemplo, a tag <meta>, usada no cabeçalho para introduzir metadados), e tags abertas, que precisam de duas: uma de abertura e outra de fechamento, utilizando a barra "/" (como as tags de texto paragrafado <p>).

Atributos de Tags

(Voltar ao topo)

As tags também possuem atributos, usados para personalizá-las, mudando sua estrutura ou funcionalidade.

Existem atributos gerais que podem ser usados por maioria das tags, como class, id, ou style. Mas também há atributos específicos para certas tags, como type para <input> e src para <img>.

Textos

(Voltar ao topo)

A tipografia de uma página web é de extrema importância para um documento HTML. Também por isso, possuem tags próprias, e abertas. Alguns exemplos são:

  • <h1& até <h6>: tags de títulos, subtítulos e outras formas de texto. Organizadas por uma hierarquia, onde <h1> é a mais importante, maior e mais grossa, e <h6> a menor delas.
  • <p>: texto automaticamente paragrafado.
  • <blockquote>: citação ou observação menor.

Existem várias outras, porém estas são algumas das mais importantes.

Listagem

(Voltar ao topo)

Existem dois tipos de listas: ordenadas (<ol>), automaticamente numeradas em ordem crescente, e não ordenadas (<ul>), listas marcadas.

Ambas delas precisam de tags filhas para itens de lista (<li>), além de serem abertas.

Links

(Voltar ao topo)

As hiperligações conectam páginas entre si, através de imagens ou textos. Como elementos fundamentais de um site, os links possuem tags para si.

A estrutura de um link HTML é <a href="local">;

  • "a" é a âncora, tag que vincula links internos e externos.
  • "href" é o atributo de referência de hipertexto. A URL do site vinculado ao nosso site. Também pode ser uma ID dento da própria página ou um outro arquivo HTML na mesma pasta.
  • Há também o atributo "target", que especifica onde será aberto o site vinculado, se será em uma nova aba (_blank) ou na mesma aba (_self).
  • O título (title) é o texto escrito para o seu link.
